static GeneratedPattern* projectPattern(Projector *projector,
Pattern *pattern)
Takes a Projector and a Pattern and, if the given projector can project the given pattern, returns a GeneratedPattern object that contains the matrix of pixel values representing the light pattern created by the projector.
